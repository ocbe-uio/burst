Introduction

This is the report for the first interim analysis of the BURST trial. The data are based on an export from the Viedoc electronic data capture at 2023-07-04 system time stamped _20230704_090943.

While the results are based on real data, the treatment allocation has been drawn randomly for this report. Thus, this is a mock-up report intended to show how the final report will look like, without showing the actual results of the trial and the treatment differences.

There were 13 included patients of which 10 are included in the full analysis set.
